## 優先度: 高(次スプリント候補)

### B-03 タグ機能 【M】
**ストーリー**: ノートを整理したいユーザーとして、ノートにタグを付けて絞り込みたい。
**受け入れ条件**:
- ノートに複数タグを付与・削除できる
- 一覧でタグによる絞り込みができる
- タグはノートデータ(S3)に保存され、デバイス間で同期される

### B-13 既存 E2E テストの修繕 【S】(技術的負債)
main 時点で8件のテストが失敗している(`locator('h2')` の strict mode violation 等)。
対象: basic-functionality ×2、dev-server-functionality ×2、final-working ×1、fixed-selectors ×3。
CI の信頼性のため早期に修繕する。

### B-04 Markdown プレビュー強化(シンタックスハイライト)【S】
